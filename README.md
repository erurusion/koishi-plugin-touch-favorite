# koishi-plugin-touch-favorite

[![npm](https://img.shields.io/npm/v/koishi-plugin-touch-favorite?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-touch-favorite)

戳一戳好感系统
</div>

## 功能
- 戳一戳根据不同好感产生不同回复.
- 每次戳一戳提升1好感度.

## 使用方法
- koishi插件市场.
